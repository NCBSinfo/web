About NCBSinfo Web and Android App

Android App
NCBSinfo is an application for regular use, especially helpful for people working at the National Centre for Biological Science (NCBS), Bangalore. Our aim is to create a mobile application, which will serve as an application-cum-database for important information pertaining to NCBS. With the help of this, we will try to reduce the time spent on searching for shuttle timings and contact numbers.

Web
NCBSinfo web is specially made for non-android users who have no access to our android app. However regular android user can also take advantage of this and can use this website to check information as well as update preferences. These preferences will be synced between android app and website.

Privacy Policy Web and Android App

Web
In this web site, we do not collect any additional personal information except information provided by you in the preferences tab. All of your personal information is stored in the secure Google server. We do not sell or distribute this information to third party unless required by law. We collect some anonymous user information like browser used, country, pages accessed etc through Google analytics. This information will be used to make better user interface and user experience. Source code used to make this website is open source and free on the GitHub . You are encourage to take a look at it and contribute to it's development if you can.

Android App
Best part of an open source app is that you will know exactly what is going on inside the app. We encourage you to check our source code on GitHub . Code is heavily commented for readability. We collect various personal and general information from this app to make a better user experience. Following is the list of all information that we get from users.

What information we collect (and why?)

• Name and email
(To authenticate you on our server)
• Default route
(To customize your app when you login again)
• Unique Token
(This token is generated by Firebase service through which you can get notifications from admins)
• Current NCBSinfo App version
(To remind you about new updates which will be free of bugs and issues)
• Last login
(To design network sync algorithms.)
• App open count
(To check active users.)
• Number of network calls done by app
(To block spammers and malicious app modifications )


If you don't want any of above information to be send to our server, you can use OFFLINE mode.
WE DO NOT COLLECT ANY PERSONAL INFORMATION FROM OFFLINE MODE USERS.

NO ONE can check or see your password, not even developers. They are sent securely with https protocol and are stored in Google's cloud database with encryption.

Google and Firebase analytics
Besides above personal information, we also receive some anonymous statistics of all users who have installed this app. This information generally includes Android Version, Default language, Country, App version, Uninstall count etc.

Crash Report
This will send us anonymous crash reports (if any) which we use to fix bugs and make app better :)
